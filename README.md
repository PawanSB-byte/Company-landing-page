## 📌 Project Overview

This project is a responsive Company Landing Page developed using HTML5 and CSS3.

The landing page represents a fictitious technology company named **NovaTech Solutions**.  
It showcases the company introduction, services offered, testimonials from clients, and contact information.

The project focuses on semantic HTML structure, modern CSS styling, responsive design, and clean code organization.

---

## 🚀 Features

- Semantic HTML5 structure
- Professional hero section
- Services displayed in card layout
- Styled testimonials section
- Contact information with UI-only contact form
- Responsive design using media queries
- Clean and well-commented code
